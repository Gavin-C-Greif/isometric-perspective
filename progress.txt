## Codebase Patterns
- Legacy v11 paths: marked with "// LEGACY: v11-only; not executed on v13"; module targets v13 so these branches never run
- Grid config: Use renderGridConfig, gridConfigUpdate, closeGridConfig hooks; call canvas.draw() after grid/ruler edits to force refresh in isometric scenes
- Use optional chaining (html.querySelector?.()) when targeting Foundry DOM elements that may vary by version
- HUD sync: canvas.hud.token/tile/drawing have .object and .element when active; canvasPan re-syncs HUD positions for isometric stage transforms
- Occlusion: PIXI removeChildren() does not destroy; explicitly destroy sprites and masks; use texture: false for shared textures; canvasTearDown for teardown cleanup
- Dynamic tile: same pattern—destroy cloned sprites with texture: false before rebuild; destroyAlwaysVisibleContainer clears lastControlledToken
- Token sort: autosorting.js = authoritative document.sort writes; token.js _refreshSort = display-only mesh.zIndex; both use calculateTokenSortValue
