# Ralph Progress Log
Started: 2026-02-24

## Codebase Patterns
- Use Node built-in test runner (`node --test`) for tests; test files use `.test.mjs` extension.
- Shared math fixtures live in `tests/fixtures/math-fixtures.js`; tolerance helpers in `tests/helpers/tolerance.js`.
- Project uses `"type":"module"` in package.json; release.cjs is CommonJS for build script compatibility.
- Pure coordinate math (cartesianToIso, isoToCartesian) in utils.js has no Foundry deps and can be tested in Node.
- Extract placement/transform math to pure functions in utils.js for unit testing; transform.js calls them with document values.
- Use safeDivide(num, denom, fallback) for any division where denominator can be 0 or non-finite.
