# Legacy Occlusion Experiments (Non-Production)

These files are alternative occlusion implementations that are **not used** in production. The module uses `scripts/occlusion.js` as the active implementation.

- **occlusion2 v15 (cpu gpu choose).js** – CPU vs GPU mode selection (CPU working but heavy; GPU not fully working)
- **occlusion2 v21 (simple test 2).js** – Different approach (not fully working)
- **occlusion3.js** – Token–token occlusion (not fully working)

Archived as part of US-006 (legacy cleanup). Module targets Foundry v13.
